Usage Guidelines

Logo Usage

  • Always maintain adequate clear space around the logo
  • Do not distort or stretch the logo
  • Use the cyan (#22d3ee) color for the logo icon
  • Logo can be used on dark or light backgrounds
  • Minimum size: 24px height for digital use

Color Usage

  • Primary background: #09172b for dark theme
  • Use gradient (cyan to violet) for primary CTAs and highlights
  • Maintain sufficient contrast for accessibility (WCAG AA)
  • Cyan (#22d3ee) for icons and interactive elements
  • Violet (#a78bfa) for primary actions and emphasis

Typography

  • Use SF Pro or system fonts for optimal performance
  • Maintain tight letter spacing for headings (-0.5px to -1.5px)
  • Body text should be minimum 16px for readability
  • Use font weight variations for hierarchy (400, 500, 700, 800)
