Logo Construction & Specifications

Icon Component

  • Size: 48px × 48px (scalable)
  • Background: Linear gradient (135deg)
  • Gradient Colors: #2563eb → #3b82f6
  • Border Radius: 8px (rounded corners)
  • Letter: "F" in white, bold (700 weight)
  • Symbol: "=" created with white dots
  • Box Shadow: 0 4px 20px rgba(59, 130, 246, 0.3)

Text Component

  • Layout: Stacked vertical (FUEL over FLUX)
  • "FUEL" Color: #3b82f6 (primary blue)
  • "FUEL" Size: 22px
  • "FUEL" Weight: 600 (semibold)
  • "FLUX" Color: White (#ffffff) on dark
  • "FLUX" Size: 20px
  • "FLUX" Weight: 500 (medium)
  • Letter Spacing: -0.3px to -0.2px (tight)

📋 Brand Usage Guidelines

✓ DO

  • Use #3b82f6 as primary brand color
  • Maintain dark background (#0f172a)
  • Use gradient for metric cards
  • Keep logo proportions consistent
  • Ensure proper contrast for text
  • Use white text on dark backgrounds

✕ DON'T

  • Don't distort or stretch the logo
  • Don't use light backgrounds for main UI
  • Don't change the blue color values
  • Don't use low-contrast text colors
  • Don't mix incompatible gradients
  • Don't alter logo proportions
